### GL C/C++ BaseCamp Testing - Technical Task
## Usage
Build the project with `make`. Then type `./file_searcher FILE_TO_SEARCH.EXTENSION ROOT_DIRECTORY`. 
Example: `./file_searcher thread.h D:\\`
